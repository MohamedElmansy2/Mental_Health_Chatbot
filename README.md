## ⚙️ System Pipeline

```
User message (any language)
     │
     ├─► Module 1 — Language Detection   (TF-IDF + LinearSVC)       → language code
     ├─► Module 2 — Emotion Classifier   (RoBERTa, HF Hub)          → emotion label
     ├─► Module 3 — Intent Classifier    (Few-shot via Groq)         → intent label
     │
     └─► intent == asking_mental_health_question?
              YES → Module 4 — RAG  (Qdrant + Sentence Transformers + Groq)  → answer
              NO  → direct canned reply, translated to user language
```

| Step | Module | Technology | Output |
|------|--------|-----------|--------|
| 1 | Language Detection | TF-IDF char n-grams + LinearSVC | `en`, `ar`, `fr`, … |
| 2 | Emotion Classifier | RoBERTa fine-tuned | `fear`, `sadness`, `joy`, … |
| 3 | Intent Classifier | Few-shot prompting (Groq) | `asking_mental_health_question`, … |
| 4 | Q&A RAG | Qdrant + Sentence Transformers + Groq | Empathetic counselling answer |

## 🚀 Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/your-username/mental-health-chatbot.git
cd mental-health-chatbot
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Set up environment variables
```bash
cp .env.example .env
```
Open `.env` and fill in your API keys:
```
GROQ_API_KEY=your_groq_key
QDRANT_URL=https://your-cluster.qdrant.io
QDRANT_API_KEY=your_qdrant_key
EMOTION_MODEL_ID=your_hf_username/emotion-roberta
LANG_MODEL_PATH=models/language_detection_tfidf_linearsvc.pkl
```

### 4. Run notebooks once (setup)
| Notebook | What it does | Where to run |
|----------|-------------|-------------|
| `02_clean_emotion_classifier.ipynb` | Train RoBERTa → push to HuggingFace Hub | Google Colab (GPU) |
| `04_rag_pipeline.ipynb` | Embed counseling dataset → upload to Qdrant | Local |

### 5. Start the API
```bash
uvicorn app:app --reload
```

### 6. Open the chat UI
Go to **http://127.0.0.1:8000**
